PersistenceService 

trait PersistenceService extends AnyRef

Service to persist JSON-like data structures. To avoid bindings to any kind of JSON library, the supported data structure is kept as Java Map.

The supported value types are: - null - java.lang.Boolean - java.lang.Long - java.lang.Double - java.lang.String - java.util.Collection - java.util.Map (with String keys)
